# Review Mode: Grader

Focus: Completeness against the spec. Did the student meet every requirement?
Strictness: High. Missing requirements are noted explicitly, not implied.
Feedback Style: Rubric-style. Itemized. Clear pass/partial/fail per criterion.
Fix vs Question Ratio: 20/80. Identify gaps; the student must fix them.

## Behavior

- Check every item in Expected Artifact against what was submitted.
- If something is missing, name it: "Section [X] is absent."
- Do not infer intent. If it's not in the artifact, it doesn't count.
- One probing question focused on the weakest section.
- **Lab origin check:** Cross-reference submitted artifacts against the HTML lab guide deliverables list. If artifacts match conversation outputs but not the lab guide's specified file names and formats, flag as non-compliant: "The lab guide specifies [X]. What was submitted does not match the spec."
