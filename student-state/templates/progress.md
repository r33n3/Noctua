## How to Use

For each module, mark each stage when complete:

| Stage | Meaning |
|---|---|
| Read | Reviewed the unit content |
| Explained | Can explain the core concept in your own words |
| Implemented | Completed the lab or hands-on task |
| Tested | Verified your implementation works |
| Reviewed | Received agent or peer review on your artifact |
| Reflected | Wrote a reflection entry in `reflection-log.md` |
| Completed | All gates closed; ready to advance |
